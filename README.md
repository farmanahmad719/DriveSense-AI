## DriveSense AI

## Overview

DriveSense AI is an AI-powered Driver Monitoring System that detects:

- Driver Drowsiness
- Yawning
- Head Pose
- Phone Usage
- Driver Attention Score

using

- OpenCV
- MediaPipe
- YOLOv8
- Python

## Features

- Adaptive EAR
- Smart MAR
- Head Pose
- Phone Detection
- Attention Score
- Dashboard
- Event Logging
